Exercício de Língua Portuguesa

Responda às perguntas abaixo com uma palavra ou uma frase curta:

1. O que é um adjetivo?

2. Dê um exemplo de um substantivo e explique por que é substantivo.

3. O que é um verbo? Dê um exemplo.

4. Como se forma uma frase? Cite os elementos básicos.

5. O que são pronomes? Dê um exemplo.
